# Rock Paper Scissors Game (C++)

This is a simple console-based Rock Paper Scissors game built using C++.

## Features
- User vs Computer gameplay
- Random number generation
- Conditional logic for winner determination
- Input handling

## Concepts Practiced
- if-else statements
- switch-case (if used)
- Random number generation
- Functions (if used)

## How to Run
1. Compile using a C++ compiler.
2. Run the program.
3. Enter your choice (Rock, Paper, or Scissors).
4. See the result!

This project was created as part of my C++ learning journey.
